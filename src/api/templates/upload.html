Hey there! Looks like you're looking forward to creating a new course on Everskill. To quote Wikipedia, "Thank you, and welcome".
You can use the [n3rdium/everskill-course-template](https://github.com/n3rdium/everskill-course-template) to get started.
Once you've created it, open up a discussion on our GitHub repo mentioning the github repo: [Course Addition Request](https://github.com/N3RDIUM/Everskill/discussions/new?category=course-addition-request)
Once the community accepts your course, we will add it to the website. You will be given a ton of coins and gems as a reward for helping the community.
